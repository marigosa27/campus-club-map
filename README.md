# Campus Club Meeting Map

Interactive qgis2web map for students to find club meeting locations on campus.

## Update Weekly Club Meetings

Edit:

`layers/clubmeetings_9.js`

Each meeting includes `club`, `buildname`, `date`, `time`, `classroom`, `info`, and point `coordinates`.
See `EDIT_CLUB_MEETINGS.md` for the copy-paste template.

## Publish With GitHub Pages

This folder can be published directly with GitHub Pages. Keep `index.html` at the repository root.
